Our Projects

VTOL Drone for Marine Mammal Surveys

A cutting-edge VTOL drone system for line transect surveys and Photo ID observation of marine mammals, focusing on Chinese white dolphin (Sousa chinensis) and finless porpoise (Neophocaena). Powered by AI-driven image recognition and eco-friendly electric propulsion, it ensures precise data collection with minimal environmental impact.

  • Technologies: Python, AI, Drone Technology
  • Features: Real-time species identification, low-carbon footprint, high-resolution imaging

Image Processing Script

A Python script for processing images, featuring error handling with OpenCV's cv2.seamlessClone and dataset management for drone-captured dolphin images (top and side views).

  • Technologies: Python, OpenCV
  • Features: Error logging, batch processing

YOLO Dataset Management

Scripts to merge and update YOLO dataset labels for AI-driven surveillance, remapping classes like 'dop_side', 'dop_top', 'ship_side', 'ship_top' and updating data.yaml.

  • Technologies: Python
  • Features: Class ID remapping, dataset configuration

Flask Web Application

A responsive Flask app with a header inspired by etaklogistics.com, featuring a leave application form with fixed apply dates and selectable leave ranges.

MS SQL Report Generator

A Python-based report generator using pyodbc, tkinter, and ReportLab, with a GUI for selecting report types and date ranges, exporting to PDF.
